Blur Enhancement

This image has undergone a blur effect to soften its appearance. The blur settings applied include a horizontal and vertical radius of 9 pixels each, which creates a smooth, out-of-focus effect. This enhancement is useful for reducing sharpness and adding a dreamy, ethereal quality to the image.

Pixelize Enhancement

The pixelize effect has been applied to this image with a width of 2 pixels and a height of 17 pixels. This transformation results in a mosaic-like appearance, where the image is broken into large, blocky pixels. This effect emphasizes geometric abstraction and can be used to create a stylized, retro look.

Edge Detection Enhancement

This image has been processed with edge detection filters, highlighting the edges and contours within the image. This enhancement emphasizes the structural details and outlines, providing a clearer view of shapes and boundaries. It is particularly useful for creating high-contrast, detailed line work.

Oil Paint Enhancement

The oil paint effect has been applied with a brush size of 3 and smoothness set to 20. This enhancement simulates the look of an oil painting, adding texture and brushstroke effects to the image. The result is a rich, artistic rendering that combines the softness of traditional painting with a digital touch.
